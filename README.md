I'm a senior software engineer specializing in backend and core engineering. For the last 10 years, I have been developing backend application. I have developed a variety of products, including designing for the Global Services platform.

I love designing systems that are light yet powerful, distributed yet synchronized and beautiful yet effective. I also frequently blog about the world behind the screen which involves system designs, databases, security, servers, optimisation and also promising technologies like Blockchain and AI.
